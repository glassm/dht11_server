# Getting Started

### Goal
* Gather data from DHT11 Temperature/Humidity sensor
* Make the data available through a REST api (http://<host>:8080/dht/dht11)

### Installation
* Uses Java JDK11
* mvn clean install
### Launch
* on command line in terminal at project root 'java -jar dht11_server.jar'
